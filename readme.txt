Professional SEO + AEO + GEO plugin for technical SEO, schema, sitemaps, bots control, and actionable audits.

== Description ==
Open Growth SEO provides practical technical SEO controls with a clean WordPress-native UX.

Features include:
- Search appearance templates and per-content overrides
- Canonical and robots controls
- Managed robots.txt with GPTBot and OAI-SearchBot controls
- XML sitemap index and post-type sitemaps
- JSON-LD schema (Organization, WebSite, WebPage, Article, Product)
- AEO and GEO checks integrated into technical audits
- Optional IndexNow integration with queue processing
- REST and WP-CLI tools for teams and agencies

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate **Open Growth SEO** from Plugins.
3. Open **Open Growth SEO** in wp-admin and run Setup Wizard.

== Frequently Asked Questions ==
= Does this guarantee rankings? =
No. The plugin provides technical controls and quality checks.

= Does this require external services? =
No. Integrations are optional.
