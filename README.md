# Open Growth SEO

Open Growth SEO is a production-focused WordPress plugin for SEO + AEO + GEO.

## Core modules
- Dashboard and setup wizard
- Search appearance and content-level controls
- Robots and canonical management
- XML sitemaps
- JSON-LD schema engine
- AEO/GEO analyzers
- Optional IndexNow integration
- REST API and WP-CLI commands
- Technical audit engine

## Quick start
1. Activate plugin.
2. Open `Open Growth SEO > Setup Wizard`.
3. Configure templates, robots, schema, and integrations.
